# article-list-details

Build an app that: 
- Lists a number of articles, each article in the list should have a headline, and image and a source
- The user should be able to click on the article which would the article details Th details view should also contain a list of relevant articles connected to that article

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/article-list-details.git
cd article-list-details
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
